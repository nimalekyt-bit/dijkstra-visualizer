# GraphLab — десктопная версия

Обёртка веб-приложения в нативное окно Windows. Подход тот же, что у AssetForge:
**Python + pywebview (Edge WebView2) + PyInstaller (onefile)**, причём сам `.exe`
является одновременно **установщиком и приложением**. Сам сайт не меняется —
десктоп просто открывает его в отдельном окне и работает офлайн.

В отличие от AssetForge здесь нет FastAPI/uvicorn и базы данных: проект
статический, файлы отдаёт встроенный `http.server`. Локальный сервер нужен
только для `fetch('data/examples.json')` (он не работает через `file://`).

## Как работает установщик

При запуске скачанного `GraphLab.exe` открывается **графический мастер установки**
(окно с собственным тёмным заголовком, боковой градиентной панелью, индикатором
шагов, анимацией переходов и прогресс-баром):

1. Приветствие.
2. О программе (краткая информация).
3. Параметры: выбор папки установки (по умолчанию `%LOCALAPPDATA%\GraphLab`,
   без прав администратора), требуемый размер и опции — ярлык на рабочем столе и
   меню «Пуск».
4. Прогресс установки с процентами.
5. Готово — кнопки «Запустить GraphLab» и «Закрыть».

Мастер копирует приложение в выбранную папку, создаёт ярлыки и регистрируется
в «Программы и компоненты». Повторный запуск через ярлык открывает уже
установленную копию без переустановки.

Удаление — стандартное, через «Параметры → Приложения» или «Программы и
компоненты» (вызывает `GraphLab.exe --uninstall`, который убирает ярлыки, запись
в реестре и каталог установки).

### Флаги командной строки

- `--silent` — тихая установка в папку по умолчанию без интерфейса (для развёртывания).
- `--uninstall` — удаление.
- `--portable` — запуск без установки.

Мастер сделан на **tkinter** (стандартная библиотека Python): надёжно работает в
сборке PyInstaller и не зависит от моста WebView2 (у pywebview js_api на этой
машине обнаружен баг с бесконечной рекурсией при сериализации окна).

## Файлы

- `run_desktop.py` — точка входа: мастер установки / локальный сервер / окно.
- `installer.py` — логика установки, тихого режима и удаления.
- `wizard.py` — графический мастер установки (tkinter).
- `ui_widgets.py` — кастомные виджеты (скруглённые кнопки, чекбокс, прогресс).
- `GraphLab.spec` — конфигурация PyInstaller.
- `make_icon.py` — генерация иконки `GraphLab.ico`.
- `build_exe.bat` — установка зависимостей, генерация иконки и сборка `.exe`.

## Запуск в режиме разработки

```bash
pip install pywebview
python desktop/run_desktop.py
```

В dev-режиме (не собранный exe) самоустановщик не активируется — сразу
открывается окно приложения.

## Сборка .exe

```bash
desktop\build_exe.bat
```

Результат: `desktop\dist\GraphLab.exe` — один самодостаточный файл (~44 МБ).
Именно его выкладывают на сайт для скачивания.

## Требования

- Windows c Edge WebView2 Runtime (на Windows 10/11 обычно уже установлен).
- Python 3.11.

## Ограничения

- Exe не подписан code-signing-сертификатом → при первом запуске возможно
  предупреждение SmartScreen. Это ожидаемо для неподписанных сборок.
